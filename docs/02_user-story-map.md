## MVP スコープ サマリー

### 🔴 Must（ハッカソン必須）— 全15ストーリー

```
オンボーディング(3) → 写真UP(3) → AI解析(2) → 動物カード(2)
→ アバター表示(1) → 短冊入力(1) → ロードマップ(2) → プロフィール(1+ログアウト)
```

### 🟡 Should（時間があれば）— 全9ストーリー

- 日記編集・一覧、動物カード共有、感情履歴、アバター感情連動
- 複数夢管理、進捗チェック、チェックイン通知、ポートフォリオ

### 🔵 Could（デモで映えるなら）— 全4ストーリー

- 月間感情サマリー、アバターインタラクション、アバター進化、祝福演出

### ⚪ Won't（将来拡張）— 全2ストーリー

- メール/パスワード認証、SNS投稿取り込み
